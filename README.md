# LLM Chatbot Study

이 레포지토리는  
**LLM / 챗봇을 기초부터 실습 중심으로 학습한 기록**입니다.

## 목표
- LLM 기본 개념 이해
- 챗봇 구조 설계
- RAG 기반 문서 질의 응답 챗봇 구현

## 학습 환경
- Python 3.10
- PyTorch (CUDA)
- VS Code
- Conda

## 진행 방식
- 주차별 학습
- 실습 코드 + 노트 기록
- 작은 프로젝트 단위로 누적
